# pyryze-util

Pyryze Infrastructure is a reactive Java library that provides scalable infrastructure primitives and utilities for building concurrent, distributed, and high-performance backend systems.

## Installation (Maven)

```xml
<dependency>
    <groupId>io.github.josh-bless</groupId>
    <artifactId>pyryze-infrastructure</artifactId>
    <version>0.1.0</version>
</dependency>
